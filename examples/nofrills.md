# Welcome to nofrills

A **minimalist** Markdown slideshow generator

* Written in *Rust*
* No external dependencies
* Just the essentials

---

## Basic Formatting

You can use **bold** and *italic* text

Code can be `inline` or in blocks:

```rust
fn main() {
    println!("Hello, slideshows!");
}
```
